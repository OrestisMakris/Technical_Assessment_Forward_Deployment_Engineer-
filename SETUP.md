### What to learn:
1. **Environment Variables**
   - They store secrets outside your code
   - Loaded at runtime from `.env` files
   - Python: `os.environ["KEY_NAME"]`
   - Shell: `export KEY_NAME=value`

2. **`.env` files**
   - Local copy: create `.env` in project root (copy from `.env.example`)
   - Git won't track it (in `.gitignore`)
   - Only you see your keys

3. **API Security**
   - Never paste keys in chat, Slack, GitHub, etc.
   - Regenerate keys if accidentally exposed
   - Use one key per environment (dev, staging, prod)

## 🧠 CONCEPTS TO UNDERSTAND

### REST APIs
- `GET` → fetch data (flights, bookings)
- `POST` → create something (book flight, add baggage)
- `PUT/PATCH` → update something (reschedule, modify booking)
- `DELETE` → remove something (cancel booking)

### FastAPI
- Modern Python web framework
- Auto-generates interactive docs at `/docs`
- Pydantic validates request/response data
- Routes are decorated with `@app.get()`, `@app.post()`, etc.

### LLMs (Large Language Models)
- Claude (Anthropic) vs Gemini (Google) - we switched to Gemini
- They take text input, return text output
- System prompts guide behavior
- Used for: customer simulation, evaluation, code generation

### Evaluation Loop
1. **Simulate** → LLM plays customer
2. **Evaluate** → score conversation 1-10, classify root cause
3. **Fix** → rewrite prompt or patch code
4. **Push** → send to ElevenLabs
5. **Repeat** → until passing or max iterations

To make it permanent (Windows):
1. Windows Settings → Environment Variables
2. New User Variable: 
   - Name: `NGROK_PATH`
   - Value: `C:\Users\orest\AppData\Roaming\Python\Python311\Scripts`
3. Add to PATH: `;%NGROK_PATH%`
